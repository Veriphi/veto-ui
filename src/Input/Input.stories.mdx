# Input

An extension of the base HTML `<input />`. Includes the exact same properties as the native element, with an
additional `variant` prop that lets you customize the outline colour.

Highly recommeneded to use this component in conjunction with the `<FormField />` component

## Addons

Inputs can also have addons attached to them. You may use the `<InputWithAddon />` component to do so.
