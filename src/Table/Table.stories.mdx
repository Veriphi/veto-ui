# Table

The `<Table />` is basically a pre-styled basic `<table />`.
It has all the same properties as the native component.

Use the native internal table elements to create your table.

If you require advanced functionalities, such as sorting, please use
[react-table](https://github.com/tannerlinsley/react-table) to add such functionalties.
