# Card

Card is used to contain elements in a defined container in a page.

A card will always span full width. The `<Card />` uses the `<Box />` component under the hood. This means that
you may override styles using the `sx` props.

<Story name="basic">
  <Card>This is a card</Card>
</Story>

<br />
<br />

## Variants

Cards have multiple variants. You may use the `variant` prop to customize the background color of the card.

Please note, that due to theme swaps and such, you will need to manually adjust the `<Text />`
