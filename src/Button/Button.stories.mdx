# Button

This component extends `<Box>`. As such, it has the `sx` props in which you can forward
whatever styles you wish.

Additionally, `<Button />` supports all normal props a regular `<button />` should support. So things like
`onClick` will work as expected.

## Variants

The `<Button />` component has a prop called `variant`. Depending on the string that is passed, the style of
the button will change

## Icons

You may add an icon to the `<Button />` via the `icon` prop. The icon being passed in must be a valid font-awesome
icon.

For example, if you wish to add the checkmark icon, you must first import the corresponding icon:

```jsx
import { faCheck } from '@fortawesome/free-solid-svg-icons/faCheck';
```

Once imported, you can add it to the Button via props.
