# Text

Text is a simple `<Box />` component with the appropriate variant key attached to it.

In order to style it properly, you will need to pass in appropriate variant string into the `variant` prop.

**Please note:** the underlying element is a `div` by default. You will need to use the `as` prop to properly
set the right element.

<Preview>
  <Story name="text">
    <div>

      <h3>Heading 1 Element</h3>
